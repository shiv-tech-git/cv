Projects

Drill controller

Drill controller is an angular speed stabilizing system for DC motor which mounts on drill machine. System uses optical sensor for measuring angular speed and PID controller for calculating value of providing to motor power. User has the ability to choose the speed via potentiometer. Current speed is displaying on 7 segment 4 digit display module.

Relay tester

Relay tester is the device which was intended to facilitate the procedure for measuring the response time of electromechanical relays. Device has 2 voltage inputs (400V max) and 2 sensitile to contacts state inputs.
